프로그램 이름 : 차빼기 퍼즐
프로그램 기능
	- 실행 시 시작 화면 출력
		- 아스키 아트로 이상한 마을 표지 띄우기?

	- 퍼즐 난이도 설정
		- 퍼즐 미리보기 이미지 AI 이미지 생성?
		- 퍼즐 난이도에 따른 문제 설정

	- 키패드로 움직일 차량 선택
		- 중복 리스트에는 차량을 차량 번호로 저장
		- 차량의 길이는 2
		- 8 : ↑
		- 4 : ←
		- 6 : →
		- 2 : ↓
		- 5 : 선택
		- 0 : 취소

	- wasd로 차량 이동
		- 중복 리스트로 현재 상태 저장
		- 특별 커맨드로 정답보기
		- 움직인 횟수 카운트
		- 앞에 장애물이 있다면 움직일 수 없고 횟수 카운트도 X
		- 연속으로 같은 방향은 횟수 카운트 X
		- 골 지점 도착시 레이튼 정답 화면 띄우기
		
	- 차량 이동 후 이동된 위치 갱신
	
	- 정답은 파일로 따로 저장
		- 정답보기 커맨드는 정답 파일을 읽어와서 화살표만 출력

	- 불러올 패키지
		- os

============================================================
함수 정의
- 표지 및 메뉴 화면, 정답 화면
	- 표시할 화면의 코드를 입력받음
	- 선택되기 전까지 무한 반복
	- 아스키 아트
	- 커맨드 출력
	- 반환값 없음

- 차량 이동 가능 판별
	- 표준 입력으로 받은 이동 방향과 차량의 방향, 이동할 방향에 장애물 유무로 이동 가능한지 판별
	- 반환값은 bool

- 차량 선택
	- 커서 위치를 변수로 선언
	- 커서와 겹쳐지는 자동차를 선택
	- 8 : ↑
	- 4 : ←
	- 6 : →
	- 2 : ↓
	- 5 : 선택
	- 0 : 취소
	- 반환값은 선택된 차량의 위치 or 취소 커맨드

- 차량 이동
	- global 중복 리스트의 좌표값 변경
	- 이동 횟수 카운트
	- 반환값은 변경된 좌표들

- 화면 갱신
	- 차량 이동 함수로 반환된 좌표들을 입력으로 받음
	- 받은 좌표들로 차량의 이미지 출력값은
	- 반환값 없음
