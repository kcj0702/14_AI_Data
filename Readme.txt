기본적인 흐름은 이렇습니다

# 1. 아두이노에서 주문한 테이블 번호(스위치 번호) 송신

# 2. 테이블 번호를 랜덤한 주문 코드로 변환하고 주문 코드를 주문 리스트에 저장

# 3. 기능: 매장 구조에 맞는 테이블에 주문 텍스트 생성, 주문 순서(추가 주문), 결제, 총 매출액, 받을금액, 받은금액

# 4. 상시로 표시될 기능 - 테이블 및 주문 표시, 총 매출액, 결제 메뉴, 주문순서 메뉴, 결제 완료 리스트 추가 - 함수 정의

포스기를 타겟으로 잡고 코드를 짰기 때문에 화면의 이미지가 중요하다고 생각합니다.
그래서 vscode의 폰트를 네이버 D2Coding로 바꾸어 한글, 숫자, 영어의 글자폭을 고정으로 맞춰주었습니다.
폰트를 다운받을 수 있는 링크입니다. - https://github.com/naver/d2codingfont (최신버전 다운받은 후, D2CodingAll 폴더 내의 폰트를 설치하고 vscode 폰트 패밀리 제일 앞에 D2Coding을 적으시면 됩니다.)

코드는 주피터로 실행하면 멈춤이 발생할 수 있기에 .py 파일로 실행해야합니다.

실행하면 매장의 테이블 정보와 커맨드 목록이 표시됩니다.

그 상태에서 아두이노 스위치를 누르면 각 테이블에 주문이 입력되고 커맨드로 결제, 주문목록, 금일 판매 내역을 볼 수 있습니다.
